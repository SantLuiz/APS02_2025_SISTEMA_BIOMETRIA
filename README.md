## Como executar

- Crie e ative um ambiente virtual (opcional).
- Instale as dependências: `pip install -r requirements.txt`.
- Execute a aplicação: `python main.py`.

Observação: a interface abre uma janela para o usuário informar o nome; em seguida, será solicitado um arquivo `.bmp` de impressão digital para validação.
